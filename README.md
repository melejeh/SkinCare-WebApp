# PlanetMel — Customer Web App

A Vaadin + Spring Boot web application for PlanetMel Skin Co. customers to browse products and place orders.

## Features
- Browse all skincare products with images
- Place orders with name, email, shipping address, and quantity
- Shared database with the admin desktop app

## Tech Stack
- Java 17
- Vaadin
- Spring Boot
- Spring Data JPA
- H2/PostgreSQL Database

## Part of the PlanetMel Platform
- [SkinCare-Website](https://github.com/melejeh/SkinCare-Website) — static frontend
- [Skincare-Admin-App](https://github.com/melejeh/Skincare-Admin-App) — desktop admin panel
